# Frontend React para Fuente Protocol

## Instalación

```bash
npx create-react-app frontend
cd frontend
npm install
```

## Estructura

```
frontend/
├── public/
├── src/
│   ├── components/
│   │   ├── TokenCreator.jsx
│   │   ├── TokenVerifier.jsx
│   │   ├── Dashboard.jsx
│   │   └── Navigation.jsx
│   ├── services/
│   │   ├── api.js
│   │   └── web3.js
│   ├── App.jsx
│   └── index.js
└── package.json
```

## Dependencias

```bash
npm install axios web3 ethers wagmi react-router-dom tailwindcss
```

## Desarrollo

```bash
npm start
```

## Build

```bash
npm run build
```
